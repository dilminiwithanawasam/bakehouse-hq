## 🚀 Get Started in 2 Minutes

### Step 1: Start Backend (Terminal 1)
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py seed_data
python manage.py runserver
```

### Step 2: Start Frontend (Terminal 2)
```bash
npm install
npm run dev
```

### Step 3: Open Browser
```
Go to: http://localhost:5173
Login: manager@bakery.com / demo1234
```

## ⚡ Quick Troubleshooting

### Backend won't start?
→ Check: Is port 8000 free?  
→ Solution: Kill process or use port 8001

### Frontend won't start?
→ Check: Is port 5173 free?  
→ Solution: Kill process or use different port

### Can't login?
→ Check: Is backend running?  
→ Solution: Make sure backend started first

### Page won't load?
→ Check: Try hard refresh (Ctrl+F5)  
→ Solution: Clear browser cache

**More troubleshooting:** See `HOW_TO_RUN.md` → Common Problems
